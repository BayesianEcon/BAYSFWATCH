# BAYSFWATCH
An R Package for Outlier Detection with Matrix-variate Data

You can install **BAYSFWATCH** from GitHub with:

```r
# Using devtools
install.packages("devtools")
devtools::install_github("BayesianEcon/BAYSFWATCH")
```
